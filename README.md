Esse arquivo traz instruções básicas de como rodar o projeto

Para rodar o backend do projeto:
`cd backend`
`npm run dev`

Para rodar o frontend do projeto:
`cd frontend`
`yarn start`

Necessário ter instalado em sua máquina:
[ ] MongoDB
[ ] NodeJS
[ ] NPM
[ ] Yarn
